### 👨‍💻 Инженер-разработчик | Бэкенд-системы и парсинг данных

Создаю высоконагруженные сервисы и инструменты автоматизации на **Rust** и **Python**. Специализируюсь на микросервисной архитектуре, работе с API и ETL-процессах.

🦀 **Стек:** Rust, Python, C#, C, C++, JavaScript

### 🚀 Ключевые проекты

*   **hh-scheduler** — распределенный планировщик задач с веб-интерфейсом.
*   **hh-server** — высокопроизводительный HTTP-сервер для обработки API-запросов.
*   **cv_analyzer** — интеллектуальная система парсинга и анализа резюме.
*   **ReportGenerator** — генератор многостраничных отчетов.
